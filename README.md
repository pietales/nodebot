## How to run this bot?
```
git clone https://github.com/willdoescode/nodebot
cd nodebot
touch login.json
```
```json
// add your token and prefix to the json file in the following format
{
"token": "your token here",
"prefix": "your prefix"
}
```
> Then add the bot to your server and you should be good to go
